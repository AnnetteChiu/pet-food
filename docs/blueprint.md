# **App Name**: Supply Chain Analyzer

## Core Features:

- Data Upload: Upload supply chain data from Royal Canin's Github repo.
- Metrics Display: Display key supply chain metrics, extracted from the dataset (such as supplier locations, transit times, inventory levels).
- AI Recommendation Engine: Provide supply chain recommendations, using a reasoning tool to flag and suggest specific areas to investigate.
- Github Integration: Support data from https://github.com/AnnetteChiu/mpms.

## Style Guidelines:

- Primary color: Soft blue (#6699CC), reminiscent of logistics and trust.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop for data visualization.
- Accent color: Teal (#368078), drawing attention to interactive elements.
- Body and headline font: 'Inter', sans-serif, known for its readability and modern aesthetic.
- Use simple, geometric icons to represent different metrics and functions.
- Dashboard layout, presenting data in cards and charts for easy analysis.
- Subtle transitions for chart updates and data filtering.
